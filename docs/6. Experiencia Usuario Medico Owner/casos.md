# Casos

### -Ubicación en la interfaz:
Selecciona "Casos" en el menú lateral.
![Dashboard](/img/img_solhub/exp.med.ct.4.1.casos/0.webp)

### -Objetivo:
Consultar y gestionar los casos de laboratorio (completados o pendientes).
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/1.webp)
### -Funciones: 

- Mostrar la lista completa de todos los casos médicos registrados, con:

 - Código y estado del caso (completo o incompleto).
 - Fecha de registro.
 - Nombre y cédula del paciente.
 - Sede del laboratorio.
 - Tipo de estudio.
 - Médico tratante.
 - Monto total.

- Opciones: Presionas donde dice “Acciones” tendrás las opciones “Ver” y “Generar”. 
![Inicio](/img/img_solhub/exp.med.rs.3.1.casos/5.0.webp)
![Inicio](/img/img_solhub/exp.med.rs.3.1.casos/5.1.webp)

### -Pasos para usar el módulo "Registros":
 
1. Selecciona el módulo "Registros"
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/1.webp)


2. Revisa la lista de casos.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/2.webp)


3. Utiliza el botón “Acciones” para ejecutar las acciones disponibles:
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/3.webp)

- Las acciones disponibles son: “Ver” y “Generar”. 
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/4.webp)

Si seleccionas la opción “Ver”, se generarán los detalles del Caso:
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/5.webp)

- Puede proporcionar Información Médica del paciente.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/6.webp)

- También proporciona la Información Adicional del paciente.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/7.webp)

- Selecciona el símbolo X para dejar de visualizar la información del caso médico.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/8.webp)


Si seleccionas la opción “Generar”, podrás generar un reporte del caso médico seleccionado. 
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/9.webp)

- Primero selecciona “Rellenar los Datos”.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/10.webp)

- Luego te enviará a otra ventana donde deberás seleccionar "Marcar como Completado", para continuar.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/11.webp)

- Despues te enviara el siguiente mensaje: 
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/12.webp)

- Tendras que esperar que se apruebe la solicitud
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/13.webp)

- Una vez aprobada la solicitud, podras tanto visualizar el documento, como descargarlo.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/14.webp)

- Selecciona "Revisar Documento", para visualizar detalles del caso.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/15.webp)

- Ejemplo:
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/16.webp)

- Selecciona "Descargar PDF" para poder visualizar el documento. 
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/17.webp)

- Te mostrara el siguiente mensaje.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/18.webp)

- Si seleccionas "Si", se procesara tu solicitud.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/19.webp)

- Y listo, podras visualizar el pdf del caso médico seleccionado.
 ![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/20.webp)

 - También tendras disponible la opción de "Enviar Caso", pero para esto
 tendras que tener un correo asociado al caso médico.
  ![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/20.1.webp)

4. Usa los Filtros de Búsqueda:


- Selecciona "Filtros" para utilizar los filtros de búsqueda:
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/21.webp)

- Los Filtros Generales existentes son: Estado de pago, Sede, rango de fecha, Filtrar por Médicos, Tipo de Examen, Estado de PDF, entre otros filtros...
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/22.webp)

- Y los Filtros por Rol son: Citología Positiva y Citología Negativa.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/23.webp)

- Una vez escogidos los filtros seleccionas "Aplicar Filtros".
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/24.webp)

- Y apareceran los resultados deseados:
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/25.webp)

- Si te equivocaste selecciona "Limpiar Filtros".
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/25.webp)

- Selecciona "Exportar" para descargar todos los documentos médicos de los casos.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/26.webp)

- Selecciona "Exportar" para descargar los documentos, si no quieres selecciona "Cancelar".
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/27.webp)


- Selecciona "Expandir", para ampliar la pantalla.
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/28.webp)

- Pero si quieres minimizar la pantalla, selecciona "Cerrar".
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/29.webp)

- También puedes usar la barra de búsqueda para encontrar casos por nombre, sede, entre otros…
![Dashboard](/img/img_solhub/exp.med.rs.3.1.casos/30.webp)
