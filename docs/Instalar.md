# Como intalar SolHub

### Requisitos:

- Un navegador web como Chrome, Microsoft Edge, Opera, Brave, entre otros...

- Tener acceso a conexion a internet.

### Pasos para instalar Solhub:

1. Abre tu navegador web favorito desde cualquier dispositivo (celular, tablet o computadora), y dirigete a la barra de busqueda.  
![Dashboard de Inicio](/img/img_solhub\0.instalar/1.webp)

2. Luego escribe la siguiente dirección en la barra de busqueda https://panel.solware.agency/
![Dashboard de Inicio](/img/img_solhub\0.instalar/2.webp)

3. Inmediata te llevara a la pagina de Solhub.
![Dashboard de Inicio](/img/img_solhub\0.instalar/3.webp)

4. Despues dirigete al icono de Configuraciones.
![Dashboard de Inicio](/img/img_solhub\0.instalar/4.webp)

5. Despues selecciona "Más herramientas".
![Dashboard de Inicio](/img/img_solhub\0.instalar/5.webp)

6. Selecciona "Anclar a la barra de tareas" para que se cree un acceso directo en la barra de tareas.
![Dashboard de Inicio](/img/img_solhub\0.instalar/6.webp)

7. Tambien puedes instalar la aplicacion web localmente en tu dispositivo, seleccionando el icono de "Instalar SolHub".
![Dashboard de Inicio](/img/img_solhub\0.instalar/7.webp)

8. Luego selecciona "Instalar".
![Dashboard de Inicio](/img/img_solhub\0.instalar/8.webp)

9. Una vez instalada la aplicacion te aparecera el siguiente mensaje:
![Dashboard de Inicio](/img/img_solhub\0.instalar/9.webp)

10. Selecciona los permisos que quieres conceder a la app, y selecciona "Permitir".
![Dashboard de Inicio](/img/img_solhub\0.instalar/10.webp)

11. Y listo, tendras instalado tu aplicacion web en tu dispositivo.
![Dashboard de Inicio](/img/img_solhub\0.instalar/11.webp)
